🔥 Firebase 設定テンプレート

✓ このテンプレートを使用して、DepenDrap Online を Firebase で動作させます。

1️⃣ Firebase Console から設定を取得

手順：
  1. Firebase Console にアクセス
  2. プロジェクトを選択
  3. 左上の歯車アイコン（⚙️）→ 「プロジェクト設定」
  4. 「アプリ」セクションで、ウェブアプリ（``）の設定をコピー

2️⃣ 設定値をコピー

Firebase Console から以下の値をコピーしてください：

設定項目 Firebase Console での名前 例
apiKey apiKey AIzaSyD1234567890...
authDomain authDomain dependrap-online.firebaseapp.com
databaseURL databaseURL https://dependrap-online-default-rtdb.asia-northeast1.firebasedatabase.app
projectId projectId dependrap-online
storageBucket storageBucket dependrap-online.appspot.com
messagingSenderId messagingSenderId 123456789012
appId appId 1:123456789012:web:abcdef1234567890

3️⃣ HTML ファイルに追加

以下の 3 つのファイルの <head> セクション内に、このコードを追加してください：

  • index.html
  • matchSetup.html
  • game.html
<script> window.FIREBASE_CONFIG = { apiKey: "YOUR_API_KEY", authDomain: "YOUR_PROJECT_ID.firebaseapp.com", databaseURL: "https://YOUR_PROJECT_ID-default-rtdb.asia-northeast1.firebasedatabase.app", projectId: "YOUR_PROJECT_ID", storageBucket: "YOUR_PROJECT_ID.appspot.com", messagingSenderId: "YOUR_MESSAGING_SENDER_ID", appId: "YOUR_APP_ID" }; </script>

4️⃣ 実際の例

Firebase Console から以下の設定を取得した場合：

const firebaseConfig = { apiKey: "AIzaSyD1234567890abcdefghijklmnopqrst", authDomain: "dependrap-online.firebaseapp.com", databaseURL: "https://dependrap-online-default-rtdb.asia-northeast1.firebasedatabase.app", projectId: "dependrap-online", storageBucket: "dependrap-online.appspot.com", messagingSenderId: "123456789012", appId: "1:123456789012:web:abcdef1234567890" };

HTML ファイルに追加するコードは：

<script> window.FIREBASE_CONFIG = { apiKey: "AIzaSyD1234567890abcdefghijklmnopqrst", authDomain: "dependrap-online.firebaseapp.com", databaseURL: "https://dependrap-online-default-rtdb.asia-northeast1.firebasedatabase.app", projectId: "dependrap-online", storageBucket: "dependrap-online.appspot.com", messagingSenderId: "123456789012", appId: "1:123456789012:web:abcdef1234567890" }; </script>

5️⃣ テスト

ローカルでテスト：
python3 -m http.server 8000

ブラウザで http://localhost:8000 にアクセス

ブラウザコンソールで確認：
  1. F12 キーを押してコンソールを開く
  2. 「Firebase 接続済み ✓」と表示されるか確認
  3. エラーが表示されないか確認

⚠️ セキュリティに関する注意

⚠️ 重要： Firebase 設定（特に apiKey）はクライアント側に含まれます。これは仕様です。本番環境では、Firebase のセキュリティルールで適切にアクセス制御してください。
