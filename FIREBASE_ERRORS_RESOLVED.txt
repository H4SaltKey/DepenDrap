================================================================================
                           ✅ 修正内容
================================================================================

【問題】
  Firebase SDK の CDN URL が ES6 モジュール形式を返していました。

【原因】
  以下の URL は ES6 モジュール形式のコードを返すため、ブラウザで
  import / export 構文が使用されていると認識され、エラーが発生していました。

  ❌ https://www.gstatic.com/firebasejs/9.23.0/firebase-app.js
  ❌ https://www.gstatic.com/firebasejs/9.23.0/firebase-database.js

【解決方法】
  Firebase SDK の Compat バージョンを使用するように変更しました。

  ✅ https://www.gstatic.com/firebasejs/9.23.0/firebase-app-compat.js
  ✅ https://www.gstatic.com/firebasejs/9.23.0/firebase-database-compat.js

【Compat バージョンとは】
  - Compat = Compatibility（互換性）の略
  - グローバル firebase オブジェクトを提供
  - 従来の Firebase SDK と同じ API を使用
  - ES6 モジュール形式ではなく、グローバルスクリプトとして動作
